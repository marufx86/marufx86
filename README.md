## About

Technical Artist and Unreal Engine developer specializing in **real-time optimization, Pixel Streaming, multiplayer systems, and pipeline automation**. I make 3D run fast and ship anywhere — and I build the tools that get teams there sooner.

- **Currently:** Unreal Engine Developer @ **Eagle 3D Streaming** · Founder of **Horizon ArchSystems**
- **Focus:** Pixel Streaming, performance optimization, multiplayer, Python tooling for Maya & Blender
- **Teaching:** free Unreal Engine tutorials on [YouTube](https://www.youtube.com/@marufx86) and step-by-step [written guides](https://marufdocs.vercel.app/)
- **Status:** Available for freelance work — remote, worldwide

## What I help teams with

| The problem | What I deliver |
| :--- | :--- |
| "Our game or 3D app runs slow." | Profiling, asset slimming, lighting & GPU fixes — up to **80% poly reduction** in production. |
| "We want people to try our 3D without installing anything." | Unreal Engine + Pixel Streaming, running in a normal browser tab. |
| "Clients can't picture the building from drawings." | Interactive architectural walkthroughs via **Horizon ArchSystems**. |
| "Our team loses hours to repetitive busywork." | Python automation for Maya, Blender & Unreal — batch tools, scan-to-engine pipelines. |
| "Multiplayer or VR keeps breaking." | Multiplayer & VR implementation, including over Pixel Streaming, plus debugging. |
